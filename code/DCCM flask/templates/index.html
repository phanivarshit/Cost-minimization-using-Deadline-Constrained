Welcome to our Application !!

Cloud storage offers scalability and convenience, but uploading large datasets can be expensive and time-consuming. Deadline Constraint Cost Minimisation tackles this challenge by creating a system that optimizes file uploads for cost and deadlines. It analyzes factors like file size, bandwidth, and pricing to find the most efficient upload strategy, potentially saving money and ensuring timely completion.

Our Objective

Our project aims to optimize the process of uploading files to the cloud while considering both deadlines and costs. It will develop a system that analyzes factors like file size, available bandwidth, and cloud storage pricing structures. The system will then determine the most cost-effective upload strategy for meeting specified deadlines. This may involve scheduling uploads for off-peak hours when costs are lower or prioritizing smaller files to ensure timely completion. Ultimately, the project seeks to minimize upload costs without compromising on deadlines.
